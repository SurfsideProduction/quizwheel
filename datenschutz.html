Grundsatz

Der Schutz Ihrer persönlichen Daten ist uns wichtig. Diese Datenschutzerklärung informiert Sie darüber, welche Daten auf quizwheel.ch erhoben werden, zu welchem Zweck und wie sie verarbeitet werden. Erhobene Daten

Auf quizwheel.ch wird das folgende Formular angeboten:

Beta-Anfrage-Formular
Felder: Name, E-Mail-Adresse
Zweck: Bereitstellung des QuizWheel-Prototyps für Testzwecke sowie Kontaktaufnahme mit Interessenten.
Übermittlung: Die Daten werden ausschliesslich über Ihr lokales E-Mail-Programm (mailto:) direkt an info@baelli.com übertragen. Es erfolgt keine Speicherung auf einem Server von quizwheel.ch.

QuizWheel-App (lokale Nutzung)

Die QuizWheel-App (QuizWheel.html) läuft vollständig lokal im Browser des Nutzers. Alle eingegebenen Inhalte (Quiz-Fragen, Antworten, Mediendateien) werden ausschliesslich im lokalen Speicher des Geräts (localStorage, IndexedDB) abgelegt. Es erfolgt keine Übertragung dieser Daten an externe Server.

Cookies und Tracking

quizwheel.ch verwendet keine Tracking-Cookies, keine Analyse-Tools (z. B. Google Analytics) und keine Werbenetzwerke. Es werden keine Nutzungsprofile erstellt.

Hosting

Die Website quizwheel.ch wird über GitHub Pages (GitHub Inc., USA) gehostet. Beim Aufruf der Website können technische Daten wie IP-Adresse und Zeitpunkt des Zugriffs im Rahmen der Server-Logs von GitHub erfasst werden. Diese unterliegen der Datenschutzerklärung von GitHub: github.com/privacy.
